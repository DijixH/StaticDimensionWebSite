# Static Site (Docker)

Just a container with a static website based on Ubuntu and Nginx. You can run it on x86 and aarch64.

## Usage

Simply run command :

`docker run --name static -d -p 80:80 thtom/staticsite`

[Live demo](https://html5up.net/dimension)
